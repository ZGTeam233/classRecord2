# classRecord
This is the HTML program to help student to remember
[如果你在找中文的自述文件请点击此处](/README.cn.md)

#### Features
-   Daily updates and check-ins
-   *Import*/*Export* data as JSON files
-   One-click *record*/*cancel record* for all
-   Pie chart for intuitive display

#### How to download
You only need to download files that start with 'record' and have the '.html' extension.
